cinemApp Documentation

To build the server-side component of a “movies” web application. The web application will provide userss with access to information about different movies, directors, and genres. userss will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL Query Parameter HTTP Method Request format Response format
Return a list of movies /movies none GET None A JSON object holding data about all the movies
Return data about a movie by title /movies/[title] /movies/:title GET None A JSON object holding data about a single movie, containing an ID
e.g.:
{
id: 1
title: "The Godfather",
genre: "crime",
director: "Francis Ford Coppola",
imageUrl: "https://upload.wikimedia.org/wikipedia/en/1/1c/Godfather_ver1.jpg"
description: "The Godfather is a 1972 American crime film."
featured: "yes"
}
Return data about a genre by name /movies/genres/[name] /movies/genres/:name GET None A JSON object holding data about a genre
e.g.: {
genre: "crime"
title: "The Godfather",
description: "The Godfather is a 1972 American crime film."
}
Return data about a director by name /movies/directors/[name] /movies/directors/:name GET None A JSON object holding data about a director
e.g.: {
name: "Francis Ford Coppola"
bio: "An American film director, producer and screenwriter.",
born: "07.04.1939"
died: "-"
}
Allow new users to register /users None POST A JSON object holding data about the registered users
{
Username: string
Password: string
Email: string
Birthday: date
}		 A response indicating whether the user was successfully registered
A JSON object holding data about the user that was registered, inculding an ID:
e.g.: {
id: 1
username: "Peter",
password: "password1",
e-mail: "peter@google.com",
birthday: "2000-01-13"
}
Allow users to update their info /users/[username] /users/:username PUT A JSON object holding data about the users update {
username: string
password: string
e-mail: string
}		 A text message indicating whether the dates was successfully updated
Allow users to add movies to their list of favorites /users/[username]/favorites/[title] /users/[username]/favorites/:title POST None A JSON object holding data about the movie whitch added to favorites and about the user who added
e.g.:
{
favorites: [title]
id: 1,
username: "PeterSmith",
password: "newPassword2",
e-mail: "petersmith@google.com",
birthday: "2000-01-23"
}
Allow users to remove movies to their list of favorites /users/[username]/favorites/[title] /users/[username]/favorites/:title DELETE None A text message indicating whether the movie was successfully removed
Allow existing users to deregister /users/[username] /users/:username DELETE None A text message indicating whether the user was successfully deregistered
